# Updating-character-chat-assistant-illegally

🚀 주요 변경 기능 및 사용 설명
1. 둥둥 떠다니던 아이콘 삭제 → 사이트 메뉴로 깔끔하게 쏙!
변경점: 기존에는 화면 구석에 파란색 돋보기 동그라미 아이콘이 계속 떠다녀서 화면을 가리곤 했죠? 이제 그 아이콘이 사라지고, 사이트 원래 화면 상단(추천답변 버튼 옆 등)에 깔끔한 전용 버튼이 추가되었습니다.
사용법: 사이트 상단 메뉴 툴바에 생긴 동그란 어시스턴트 버튼을 누르면 도우미 창이 열리거나 닫힙니다.
2. 창 마음대로 움직이고 크기 조절하기
변경점: 예전에는 도우미 창이 아이콘 근처에 고정되어 있었지만, 이제는 컴퓨터의 윈도우 창처럼 마음대로 위치를 옮기고 크기도 조절할 수 있습니다.
사용법:
이동: 어시스턴트 창 맨 위쪽(제목이 있는 회색 바)을 마우스로 잡고 끌면 원하는 곳으로 이동합니다.
크기 조절: 창의 맨 오른쪽 아래 모서리를 잡고 끌면 창을 크게 늘리거나 작게 줄일 수 있습니다. (설정한 위치와 크기는 다음번에도 그대로 유지됩니다!)
3. 다크 모드 완벽 지원 및 예뻐진 디자인
변경점: 투박했던 버튼과 텍스트(⚙, ✕ 등)가 **세련된 그림 아이콘(SVG)**으로 바뀌었습니다. 또한 사용자가 사이트를 '다크 모드(어두운 화면)'로 쓰면 도우미 창도 알아서 까맣게, '라이트 모드'로 쓰면 하얗게 색상이 완벽하게 맞춰집니다.
4. 엔터(Enter) 키로 바로 전송하기 & 한글 씹힘 방지
변경점: 예전에는 실수로 전송되는 것을 막기 위해 Ctrl + Enter를 눌러야 질문이 보내졌습니다. 이제는 일반 채팅처럼 Enter만 누르면 바로 질문이 전송되도록 훨씬 편하게 바뀌었습니다.
사용법:
질문 전송: Enter
줄바꿈(엔터 치기): Shift + Enter
참고: 한글을 치는 도중에 엔터를 눌러도 글자가 무시되거나 두 번 보내지는 버그도 함께 고쳐졌습니다.
5. 한 캐릭터 안에서 여러 개의 대화방(탭) 나누기
변경점: 가장 강력한 추가 기능입니다! 예전에는 캐릭터마다 질문/답변 기록이 하나로만 쭉 이어졌습니다. 이제는 하나의 캐릭터 안에서도 여러 개의 새 대화방을 만들어서 질문을 따로따로 관리할 수 있습니다.
사용법: 질문 입력창 바로 위에 있는 [새 대화방(+)] 버튼을 누르면 새로운 대화 기록을 시작할 수 있습니다. 옆에 있는 연필 모양 아이콘으로 대화방 이름을 바꾸거나, 쓰레기통 아이콘으로 삭제할 수 있습니다.
6. 나만의 '시스템 프롬프트(AI 성격/지시문)' 여러 개 저장하기
변경점: AI에게 내리는 기본 지시문(시스템 프롬프트)을 예전에는 한 개만 적어둘 수 있었지만, 이제는 여러 개를 프리셋(저장 슬롯)으로 만들어두고 상황에 맞게 골라서 쓸 수 있습니다.
사용법:
설정(톱니바퀴) 창에 들어가면 프롬프트를 추가(+), 수정, 삭제할 수 있는 기능이 생겼습니다.
전투용, 로맨스용, 요약용 등 지시문을 여러 개 만들어두고 메인 화면에서 간편하게 클릭 한 번으로 바꿔가며 쓸 수 있습니다. 프롬프트를 변경하면 다음 응답부터 반영됩니다!
7. 보이지 않는 개선: 데이터 저장 방식이 훨씬 튼튼해짐
변경점: 예전에는 브라우저 확장프로그램의 임시 저장소를 써서 가끔 데이터(대화 기록 등)가 날아가는 불안함이 있었습니다. 이제는 브라우저의 내장 데이터베이스(IndexedDB)를 사용하도록 업그레이드되어 내 대화 기록과 설정이 훨씬 안전하게 보관됩니다.
